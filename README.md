# Pet Care App

## Overview
A comprehensive mobile application for managing pet health records, care team coordination, and insurance management. Built with React Native, Firebase, and TypeScript.

## Getting Started

### Prerequisites
- Node.js 16+
- Yarn
- Expo CLI (`npm install -g expo-cli`)
- iOS Simulator or Android Emulator

### Environment Setup
1. Clone the repository
2. Copy `.env.example` to `.env`
3. Update `.env` with your configuration values:
   - Firebase credentials
   - Sentry DSN
   - Google Maps API key
